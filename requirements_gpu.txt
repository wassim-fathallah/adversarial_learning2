# GPU environment for AADA (.venv_gpu) — Python 3.11 + CUDA 12.4
#
# Same stack as requirements.txt, but PyTorch is the CUDA 12.4 build so training
# runs on the GPU. Versions are pinned to what this project was developed on.
#
# Setup on a CUDA 12.4 machine (Python 3.11 must be installed):
#   py -3.11 -m venv .venv_gpu          # Windows
#   .venv_gpu\Scripts\activate          # (Linux/macOS: source .venv_gpu/bin/activate)
#   python -m pip install --upgrade pip
#   pip install -r requirements_gpu.txt
#
# Notes:
#   - The --extra-index-url line pulls the +cu124 torch wheels (Python 3.11 only;
#     these wheels are not published for 3.13/3.14).
#   - No CUDA GPU? Use requirements.txt instead (CPU PyTorch), no index URL needed.
#   - Ollama is still required for the LLM step (see README) — not a pip package.
